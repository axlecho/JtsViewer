JtsViewer
===

这是一个 [吉他社](http://www.jitashe.org/) Android 平台的浏览器。

Screenshot
---
![index-scene](screenshot/index-scene.png)

Build
---

Windows

    > git clone https://github.com/seven332/EhViewer
    > cd EhViewer
    > gradlew app:assembleDebug

Linux

    $ git clone https://github.com/seven332/EhViewer
    $ cd EhViewer
    $ ./gradlew app:assembleDebug

其他

    使用Android Stdio导入

生成的 apk 文件在 app\build\outputs\apk 目录下
